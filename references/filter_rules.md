# 文献过滤与排序规则

本文档定义文献类型推断、等级回填和排序的详细规则，供 `tools/sort_papers.py` 和 Phase 4 数据清洗阶段使用。

---

## 1. 文献类型推断规则

### 1.1 OpenAlex 文献

根据 OpenAlex API 返回的 `type` 字段进行推断：

| OpenAlex `type` 值 | 推断结果 |
|---|---|
| `article` | Journal |
| `proceedings-article` | Conference |
| `conference-paper` | Conference |
| `book-chapter` | Book Chapter |
| `dissertation` | Thesis |
| `preprint` | Preprint |
| `dataset` | Dataset |
| 其他或缺失 | Unknown |

### 1.2 arXiv 文献

arXiv 不提供文献类型信息，统一标注为 `Unknown`。

---

## 2. 等级回填规则

### 2.1 期刊等级回填

对 OpenAlex 返回的文献，根据其 `primary_location.source.display_name`（期刊名称）匹配中科院分区表，回填等级：

1. 精确匹配 `期刊名称` 字段。
2. 若精确匹配失败，尝试忽略大小写匹配。
3. 若仍匹配失败，标注为"未收录"。

匹配成功后，将 `分区` 值转换为中文标签：

| 分区值 | 中文标签 |
|---|---|
| 1 | 中科院一区 |
| 2 | 中科院二区 |
| 3 | 中科院三区 |
| 4 | 中科院四区 |

### 2.2 CCF 等级回填

根据文献的期刊/会议名称匹配 CCF 分区表中的 `简称` 或 `全称` 字段：

1. 优先用 `全称` 精确匹配。
2. 若失败，尝试用 `简称` 精确匹配。
3. 若失败，尝试忽略大小写匹配。
4. 若仍匹配失败，标注为"未收录"。

匹配成功后，将 `分类` 值转换为标签：

| 分类值 | 标签 |
|---|---|
| A | CCF A 类 |
| B | CCF B 类 |
| C | CCF C 类 |

### 2.3 双系统标注

若文献同时在中科院分区表和 CCF 分区表中匹配成功，两个等级均保留。例如：`中科院一区 / CCF A 类`。

## 3. 排序规则

### 3.1 排序优先级原则

**符合等级要求的文献始终排在其他文献之前**，无论用户选择何种排序标准。

具体实现：先将文献分为两组，各组内按用户指定标准排序，然后拼接——符合等级要求的组在前，其他组在后。

### 3.2 按年份排序（降序）

- 按 `year` 字段数值降序排列。
- `year` 为 `null` 或 `"N/A"` 的文献排在同组末尾。
- 同年份的文献按原始相关性顺序排列。

### 3.3 按引用量排序（降序）

- 按 `cited_by_count` 字段数值降序排列。
- `cited_by_count` 为 `null` 的文献（arXiv 文献默认为 null）排在同组末尾。
- 同引用量的文献按年份降序作为二级排序。

### 3.4 按等级排序（降序）

#### 期刊等级排序权重

| 等级 | 排序权重 |
|---|---|
| 中科院一区 | 4 |
| 中科院二区 | 3 |
| 中科院三区 | 2 |
| 中科院四区 | 1 |
| CCF A 类 | 4 |
| CCF B 类 | 3 |
| CCF C 类 | 2 |
| 未收录 | 0 |

- 同时有中科院分区和 CCF 分类的文献，取两者中较高的权重。
- 同权重的文献按引用量降序作为二级排序。
- `null` 等级信息的文献排在同组末尾。

---

## 4. 去重规则

### 4.1 DOI 去重（优先）

两篇文献若 DOI 非空且相同，判定为重复。保留策略：

| 场景 | 保留版本 | 标记 |
|---|---|---|
| arXiv + OpenAlex 均存在 | 保留 OpenAlex 版本（元数据更丰富） | 来源：arXiv & OpenAlex |
| 同一来源内重复 | 保留第一条 | 来源不变 |

### 4.2 标题去重（备选）

当 DOI 为空时，按标题精确匹配去重（忽略大小写和首尾空格）。保留策略同上。
